Radar Control Program Name:
==========================
noopscan

Control Program ID (CPID):
=========================
N/A

Parameters:
==========
nbeams: N/A
intt: N/A
scan: N/A
ngates: N/A
frang: N/A
rsep: N/A

Description:
===========
noopscan is used to turn off a channel for a specified period of
time, and does not produce any data.
