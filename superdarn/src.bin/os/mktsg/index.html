mktsg

Syntax
mktsg --help
mktsg [options]
Synopsis

Determines the timing sequence for a given pulse sequence and writes it to standard output.

Options
--helpprint the help message and exit.
--versionprint the RST version number and exit.
-tauscanuse the 11-pulse tauscan sequence.
-longpulseuse the extended 16-pulse sequence.
-olduse the old 7-pulse sequence.
-baud nbaudset the number of baud in the Barker phase code to nbaud (default is 1, ie no phase coding). Valid options are 1,2,3,4,5,7,11,13.
-rsep rsepset the range resolution to rsep km (default is 45).
-nrang nrangset the number of range gates to nrang (default is 75).
-frang frangset the distance to the first range gate to frang km (default is 180).
-mpinc mpincset the multipulse increment to mpinc microseconds (default is 1500).
