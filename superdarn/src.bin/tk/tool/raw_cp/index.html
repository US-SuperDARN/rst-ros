raw_cp

Syntax
raw_cp --help
raw_cp rawacfnames...
raw_cp -old rawname
Synopsis

Determines the program identifier numbers in rawacf or raw (dat) format files.

Options
--helpprint the help message and exit.
--versionprint the RST version number and exit.
rawacfnamesfilenames of the rawacf format files. If this is omitted the file is read from standard input.
-oldfiles are in the raw (dat) format.
rawnamefilename of the raw (dat) format file.
Description

Determines the program identifier numbers in rawacf or raw (dat) format files.

The total number of program identifiers followed by a list of the identifier numbers is written to standard output.

Examples
raw_cp -old 20030408.gbr.raw

Find the program identifier numbers in the raw file "20030408.gbr.raw".
