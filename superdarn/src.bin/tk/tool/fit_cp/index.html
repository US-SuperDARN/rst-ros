fit_cp

Syntax
fit_cp --help
fit_cp [fitacfnames...]
fit_cp -old fitname
Synopsis

Determines the program identifier numbers in fitacf or fit format files.

Options
--helpprint the help message and exit.
--versionprint the RST version number and exit.
fitacfnamesfilenames of the fitacf format files. If this is omitted the file is read from standard input.
-oldfiles are in the fit format.
fitnamefilename of the fit format file.
Description

Determines the program identifier numbers in fitacf or fit format files.

The total number of program identifiers followed by a list of the identifier numbers is written to standard output.

Examples
fit_cp -old 20030408.gbr.fit

Find the program identifier numbers in the fit file "20030408.gbr.fit".
